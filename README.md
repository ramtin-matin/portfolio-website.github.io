# Ramtin Matin — Developer Portfolio

A minimal, responsive developer portfolio built with **React**, **Vite**, and **Tailwind CSS**.  
It showcases my background, technical skills, and selected projects using smooth transitions, dark mode, and clean UI components.

---

### Tech Stack
- React  
- Tailwind CSS  
- Lucide Icons  
- Vite
